`Version` **1.0.1**

`What's New`

- 修復localStorage無法儲存的問題。

---

### 📌 這是什麼

身為日誌稽核承辦人，你是否有以下困擾？
- _逐筆核對勤務表，好麻煩！_
- _查詢理由好含糊，還要比對工作記錄簿！_

沒錯！為了解決以上痛點，本人開發了這個小工具，幫助您快速比對每一筆查詢紀錄的勤務項目及工作紀錄！

### ⭐ 提前準備
你需要先從系統中匯出以下Excel檔案：
- **員警工作紀錄簿**
- **主管稽核調閱**

### ▶️ 如何使用
1. 打開程式
2. 選擇上述兩個檔案
3. 按下「開始比對」

恭喜你！現在只要查看最終生成的檔案， 就能清楚知道同事在搞什麼鬼(X

### ⚠️ 注意事項
- 工作紀錄簿單次只能查詢1000筆資料，通常無法涵蓋整個月的紀錄，請分別輸出不同時段（如每10天一個檔案），再逐一分別比對；
或者預先把所有紀錄複製到同一個檔案中，再開始比對。
- 輸出的格式為CSV檔，使用Excel開啟，可以透過「**全選**->**常用**->取消**自動換行**」
的方式加速瀏覽。
- 如果您有任何問題或意見，歡迎到[鴿手](https://pigeonhand.tw/feedback/web?option=2)中回饋讓我知道！
