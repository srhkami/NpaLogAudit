# 日誌稽核小鴿手

### 這是什麼

身為日誌稽核承辦人，你是否有以下困擾？
- _逐筆核對勤務表，好麻煩！_
- _查詢理由好含糊，還要比對工作記錄簿！_

沒錯！為了解決以上痛點，本人開發了這個小工具，幫助您快速比對每一筆查詢紀錄的勤務項目及工作紀錄！

### 需要的檔案
您需要先從系統中匯出以下Excel檔案：
- **員警工作紀錄簿**
- **主管稽核調閱**

### 如何使用
1. 打開程式
2. 選擇上述兩個檔案
3. 按下「開始比對」

恭喜你！現在只要查看最終生成的檔案， 就能清楚知道同事在搞什麼鬼(X

如果您有任何問題或意見，歡迎到[鴿手](https://pigeonhand.tw/feedback/web?option=2)中回饋讓我知道！
